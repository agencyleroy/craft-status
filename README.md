# Craft Status plugin for Craft CMS 3.x

This plugin will enable status querying for Agency Leroy dashboard.

## Requirements

This plugin requires Craft CMS 3.2.0 or later.

## Installation

To install the plugin, follow these instructions.

1. Open your terminal and go to your Craft project:

        cd /path/to/project

2. Then tell Composer to load the plugin:

        composer require craft-status/craft-status

3. In the Control Panel, go to Settings ‚Üí Plugins and click the ‚ÄúInstall‚Äù button for Craft Status.
